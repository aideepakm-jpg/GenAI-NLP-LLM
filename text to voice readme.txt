Automated Text-to-Voice Converter using pyttsx3 Library

Developed by DEEPAK.M AI Generalist IPEC Solution Private Limited Bangalore.

✅ Step 1: Install Python
Make sure Python is installed on your system.

Download from: https://www.python.org
While installing, ✔️ tick “Add Python to PATH”
Check installation:

python --version
✅ Step 2: Install Required Library
Open Command Prompt / Terminal and run:
pip install pyttsx3

✅ Step 3: Install Jupyter Notebook (if not installed)
pip install notebook

✅ Step 4: Open Your Project File
Navigate to your file location:
cd path_to_your_file
Example:
cd Downloads
Now run:
jupyter notebook

👉 This will open a browser automatically

✅ Step 5: Open Your File
Click on Text2Audio.ipynb
Notebook will open

✅ Step 6: Run the Code
Inside the notebook:
Click on the code cell
Press ▶️ Run button OR press:
Shift + Enter

✅ Step 7: Hear the Output 🎧
Once you run:
Your system will speak the text aloud
Make sure:
Volume is ON 🔊
Speakers/headphones are connected
