✅ Step 1: Install Python
Make sure Python is installed (3.8+ recommended)
Check:
python --version

✅ Step 2: Install Required Libraries
Most chatbot notebooks use these libraries. Run this once:
pip install notebook transformers torch nltk scikit-learn
👉 If your code uses extra libraries (like gradio, flask, etc.), install them too:
pip install gradio flask

✅ Step 3: Open Project Folder
Go to the folder where your file is saved:
cd path_to_your_file
Example:
cd Downloads

✅ Step 4: Start Jupyter Notebook
jupyter notebook
👉 A browser will open automatically

✅ Step 5: Open Your File
Click 👉 ChatBotAdvanced.ipynb
Notebook will open\
